#  Week 4 – Forms & User Input (Flutter)



##  About This Repository
This repository contains all Flutter applications developed during **Week 4** of the NeuroApp Internship.  
The main focus of this week was to work with **Forms, User Input, Date & Time Pickers, and Media Handling**.

These tasks help build a strong foundation in Flutter for creating real-world applications.

---

## Task 4.1 – Form Validation App (form_validation_app)

This app provides a complete **Registration Form** that validates user input before submission.

### Features:
- Name, Email, Phone and Password fields
- Email format validation
- Phone number length check
- Password security using `obscureText`
- Custom input decorations
- Error messages for invalid data
- Snackbar shown on successful submission
- Form is cleared after submission

---

##  Task 4.2 – Input Widgets App (input_widgets_app)

A **survey form** that demonstrates multiple input widgets.

### Features:
- Checkbox for selecting interests
- Radio buttons for gender selection
- Switch for notifications
- Dropdown for country selection
- Slider for selecting age
- Summary display of all selected values

---

##  Task 4.3 – Date & Time Picker App (datetime_picker_app)

An **appointment booking** app using date and time pickers.

### Features:
- DatePicker and TimePicker
- Past dates are blocked
- Date formatting using `intl`
- Calculates days until appointment
- Clean and styled UI using cards

---

##  Task 4.4 – Media Picker App (media_picker_app)

This app allows users to pick images and documents.

### Features:
- Pick image from camera and gallery
- Display selected image
- Show image
- Pick PDF, Word or other documents
- Display file
